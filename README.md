                      # Adventure-Game-using-AI
Interactive "Choose Your Own Adventure" Game

An advanced Python-based interactive adventure game featuring multiple story paths, inventory management, health system, random events, and save/load functionality.

This project demonstrates Artificial Intelligence concepts, game logic, and advanced Python programming.
 
      #Features

Interactive Story Gameplay

Health System

Inventory System

Random Events

Multiple Locations

Multiple Endings

Save and Load Game

Typewriter Animation

Replay Option
